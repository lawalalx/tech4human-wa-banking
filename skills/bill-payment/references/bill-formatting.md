# Bill Payment Formatting Rules

All responses MUST be WhatsApp-safe.

---

## Success Format

✅ Payment Successful

Biller: DSTV
Amount: ₦10,000.00
Reference: BILL-XXXXXX

---

## Formatting Rules

- Always use ₦ currency format
- Always use comma separators
- Always show 2 decimal places
- NEVER use markdown tables
- NEVER return JSON
- NEVER expose internal IDs
