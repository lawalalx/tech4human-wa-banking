# Balance Enquiry Response Formatting

Always display balances clearly and professionally.

---

# REQUIRED FORMAT

✅ Available Balance

Account: XXXXXXXX1234
Available Balance: ₦125,450.75

---

# FORMATTING RULES

- ALWAYS use ₦ currency symbol
- ALWAYS include comma separators
- ALWAYS show 2 decimal places
- ALWAYS mask account numbers except last 4 digits
- NEVER expose internal IDs
- NEVER use markdown tables
- NEVER use JSON responses
- Keep responses short and WhatsApp-friendly

---

# GOOD EXAMPLE

✅ Available Balance

Account: XXXXXXXX4581
Available Balance: ₦245,000.00

---

# BAD EXAMPLES

❌ Account Number: 2039485730
❌ Balance = 245000
❌ {"balance":245000}
❌ Markdown tables
